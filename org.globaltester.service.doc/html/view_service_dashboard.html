GT Service Dashboard

GlobalTester contains (depending on installed features) several background services, e.g. for remote control or simulation of chip cards. The status of these services can be monitored (or changed) through the optional GT Service Dashboard. This view is not visible by default but can be activated from the Window menu just like any other view.

Note: While this view allows modification of the status of several services keep in mind that most of these services can be configured in the preferences and in general no manual modification here is needed. For example all testcases delivered by secunet automatically start their required simulator automatically. If you interfere with this automatically started simulators expect your test to fail.
